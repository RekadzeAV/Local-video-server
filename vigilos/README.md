# Vigilos Community Edition 2.0

Открытая (AGPLv3) кроссплатформенная VMS с гибридным стеком: Go ядро, C++ аналитика, React веб, Qt десктоп, KMM мобильные клиенты. Проект рассчитан на сообщество и интеграторов, поставляется в нескольких редакциях с программными ограничениями.

## Миссия
- Бесплатная альтернатива коммерческим VMS для некоммерческого применения.
- Открытая платформа для разработчиков и интеграторов.
- Управляемая сообществом разработка с прозрачной лицензией и CLA.

## Лицензирование
- AGPLv3 + CLA + оговорка Non-Commercial для редакций Community.
- Требование открытия исходников производных работ.
- Механизм контроля: валидация редакций, feature gates, юридические соглашения.

## Основные инновации
- Единая кодовая база для 6 редакций, ограничения — программно.
- Гибридная архитектура (Go + C++ + React + Qt + KMM).
- Нативные интеграции с NAS (Synology/QNAP/Asustor), статическая сборка.
- Спец. редакция RegViD с готовой ОС под dedicated-железо.

## Структура репозитория (высокоуровнево)
- `cmd/` — сервисы ядра и утилиты (core, validator, migrate).
- `internal/` — доменные пакеты ядра (camera, media, storage, events, api, licensing).
- `analytics/` — C++ модуль детекции/ANPR с CGO/gRPC мостом.
- `web/` — React + TS фронтенд (RTK Query, Video.js, MUI).
- `desktop/` — Qt 6 клиент.
- `mobile/` — KMM shared + Android/iOS UI.
- `infra/` — скрипты сборки, Docker, packaging, CI/CD, лицензирование.
- `docs/` — архитектура, API, гайды, legal.

Детали см. в `docs/`.
