# Web (React + TypeScript)

Planned stack: Vite, React 18, TS, Redux Toolkit/RTK Query, MUI, Video.js, React Router v6.

Structure (planned):
- `src/app/store` — RTK store, slices, api.
- `src/app/router` — routes.
- `src/features/live-view` — grid, VideoPlayer, PTZ.
- `src/features/archive` — timeline, calendar, playback.
- `src/features/admin` — cameras, users, storage, integrations.
- `src/features/maps` — OSM/Leaflet integration.
- `src/themes` — default + NAS flavors.
- `src/shared` — components/hooks/utils.
