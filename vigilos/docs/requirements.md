# Требования и метрики

## Технические метрики (целевые)
- Задержка live: < 2 c.
- CPU: < 50% на 8×1080p.
- RAM: < 1 ГБ на 8×1080p.
- Старт: < 30 c.
- Uptime: 99.9%, восстановление < 60 c, потеря кадров < 0.1%.

## Пользовательские метрики
- 1000+ установок в первый месяц, 500+ активных через 3 мес.
- 10+ внешних контрибьюторов.
- Рейтинг 4.5/5, time-to-first-video < 10 минут, <5 обращений/день.

## Поддерживаемые протоколы
- RTSP (H.264 Baseline/Main/High, H.265 Main; AAC/G.711; TCP/UDP).
- ONVIF: профили S/G/T, Digest/WSSE.
- Производители: Hikvision, Dahua, Axis, Ubiquiti, Reolink, Tapo, EZVIZ, любые совместимые RTSP/ONVIF.

## Сетевые требования
- Входящие: 80/443, 554, 50051, 5432 (если внешняя БД).
- Исходящие: SMB 445, NFS 2049.

## Безопасность
- Шифрование архивов (AES-GCM) в планах.
- Маскирование PII в логах, secure defaults.
- security.txt для релизов.
