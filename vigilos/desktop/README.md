# Desktop (Qt 6)

План: gRPC клиент к ядру, MDI интерфейс, видеостена с drag-n-drop, пресеты раскладок, нативные адаптеры под Win/macOS/Linux.

Каталоги:
- `src/core` — API client, Models, Settings.
- `src/ui` — MainWindow, VideoWall, CameraManager, EventViewer.
- `src/platform` — платформенные обертки.
- `resources` — layouts, icons.
- `translations` — QTranslator файлы.
